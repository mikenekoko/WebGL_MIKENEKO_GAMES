### 使用技術

* C#

### 使用素材
* ドット絵 : ヤマンチュゲーム研究所様（ http://www.ob2.aitai.ne.jp/~owl/material/dot.html ）
* その他フリーBGM

# 操作方法
* a or ← : 左移動
* d or → : 右移動
* w or ↑ : ジャンプ
* j : 攻撃

# ルール
* ステージ2面構成
* スコアを集めて、ゴールを目指そう！
